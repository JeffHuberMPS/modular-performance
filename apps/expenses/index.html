How Sync Works
This tracker reads your Google Sheet's Log tab and rebuilds your dashboard from it — a second view of the same data.
Flow:
Tracker opens
  ↓
Reads your published Log (CSV)
  ↓
Builds entries from Date / Merchant / Amount / Category
  ↓
Dashboard renders fresh data
Your Wells Fargo → Make → Sheet automation is untouched. This just reads the result.
Setup: Sheet → File → Share → Publish to web → Log tab → CSV → paste that link above.
